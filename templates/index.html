What is this?

This is the Video Game API. A small project developed by Joshua Blewitt. This API allows authenticated users to get, add and delete video games to a database.

The API is free to use!

If you want to see the repo and take a look at the code, click here.

The repo also includes a Postman JSON file of example calls to the API.

PLEASE NOTE: This is a project designed for myself to practice my development skills, please DO NOT add any sensitive or personal information!

The API was developed by using the following

Backend
  • Python
    • Flask framework
    • Flask JWT Extended

Frontend

How to use the API

Authentication

The API uses JWT for authentication. At the moment, there is no way to register for an account. There is a hard coded credential to use to authenticate a session.

Please use the following JSON:

{ "username": "Josh", "password": "mypass" }

This can be used to access the /login endpoint

This will return an access token that can be used with the rest of the API

Display games

To display all games in the database, you will need to use the /displayGames endpoint. Submit a GET request to this endpoint and you will see a list of all the games in the database.

Example response:

{ "Games": [ { "Game": { "Genre: ": "Fighting", "ID: ": 1, "Name: ": "X-Men Vs Street Fighter", "Platform: ": "Sega Saturn", "Publisher: ": "Capcom", "Year": 1997 } }, { "Game": { "Genre: ": "Platformer", "ID: ": 2, "Name: ": "Super Mario 64", "Platform: ": "Nintendo 64", "Publisher: ": "Nintendo", "Year": 1996 } }, { "Game": { "Genre: ": "Racing", "ID: ": 3, "Name: ": "Wave Race 64", "Platform: ": "Nintendo 64", "Publisher: ": "Nintendo", "Year": 1996 } } ] }

Add game

To add a game to the database, you will need to use the /addGame endpoint and submit a POST request.

The endpoint accepts JSON in the following structure:

{ "name": "Wave Race 64", "platform": "Nintendo 64", "publisher": "Nintendo", "genre": "Racing", "year": 1996 }

The API will check if the year provided is in an integer format and if the request has values that aren't null. The API will return a 400 status code if the request doesn't follow the format.

Delete a game

To delete a game from the database, you will need to use the /deleteGame endpoint and submit a DELETE request.

The endpoint accepts JSON in the following structure:

{ "id": 1 }

Find a game

To find a game (or games), you will need to use the /findGame endpoint and submit a GET request.

The endpoint accepts JSON in the following structure:

{ "name": "64" }

The API will return multiple results if the search query matches multiple game names with a status code of 200. Or a 404 status code if no results are found.

Update a game

If you want to update a game, you will need to use the /updateGame endpoint and submit a PUT request.

The endpoint accepts JSON in the following structure:

{ "id": 10, "field": "name", "value": "Wave Race: Blue Storm" }

The field parameter will accept one of the following:

  • name
  • platform
  • publisher
  • genre
  • year

The API will check to see if the field is valid. It will return a status code 200 if the request was successful or a status code 400 if the field entered was invalid. If no results are found a 404 status code will be returned.
